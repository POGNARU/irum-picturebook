# irum-picturebook

김이루미 작가의 첫 그림책 **「나는야 고양이!」** 협업 아카이브입니다.
글 햇살 · 그림 김이루미 · 포근나루 편집부

공모전(창비 그림책상) 출품을 목표로, 기획·작업 일정·콘티를 공개로 공유합니다.

## 페이지

| 파일 | 내용 | 공개 주소 |
|------|------|-----------|
| `index.html` | 협업일지 — 6개월 로드맵, 24스프레드 진행 체크리스트, 작업 페이스 가이드 | https://pognaru.github.io/irum-picturebook/ |
| `conti.html` | 그림책 콘티 — 24스프레드별 글 위치·구도 가이드(SVG)·균형 메모, 전체 흐름 리듬 | https://pognaru.github.io/irum-picturebook/conti.html |

## 콘티 보는 법

- 각 칸은 펼친 책 한 장면(스프레드)이에요. 왼쪽 L 페이지 + 오른쪽 R 페이지.
- 노란 박스 = 글이 들어갈 자리, 도형 = 대략적인 구도 가이드(실제 작화는 작가 자유).
- ■ 표시는 양면 펼침 장면, 테라코타 테두리는 클라이맥스(#14 크기 역전, #21 나방).

## 원고

- 원작: 「나는야 고양이」 (햇살)
